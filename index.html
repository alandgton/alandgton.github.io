Welcome to Comp Sci 162

What is CS 162?

CS 162 is UC Berkeley's Operating Systems & Systems Programming upper-division computer science course.

Over the span of 15 weeks, students work in teams of 4 to design and implement features for PintOS. Check out the course website at cs162.eecs.berkeley.edu.

What did you do?

I taught two 1-hour discussion sections every week of 64 students total. I also graded all exams and student design docs for each in-class project (user programs, priority scheduler, and inode-based file system).

I created my own slides and linked them all here (some material sourced from lecture slides, old discussion slides from previous TAs, the internet, and the textbook).

In Fall 2019 and Spring 2020, I was in charge of the OS networking assignment. I wrote the skeleton/solution code and autograder, helped students in my office hours, and answered questions on Piazza. You can view the spec here. Students were to implement an HTTP server to serve files and handle proxy requests. Given the basic single-threaded single-process server, students were also instructed to extend server functionality to be multi-process and multi-threaded. Afterwards, students were asked to use the Apache Bench (ab) utility to load test and measure the performance of each server implementation.
